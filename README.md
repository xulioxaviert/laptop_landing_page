# Laptop Landing Page

Static and responsive landing page to promote high-end laptop deals, including a contact form.

## Features
- Modern and responsive design
- Featured products section
- Optimized images
- Functional contact form
- Easy to customize

## Project Structure
- `index.html`: Main page
- `style.css`: Custom styles
- `img/`: Images used

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/xulioxaviert/laptop_landing_page.git
   ```
2. Open the `index.html` file in your browser.

## Customization
- Change the images in the `img/` folder.
- Modify the styles in `style.css`.

## CI/CD
This project uses GitHub Actions to automatically deploy the site from the `master` branch to GitHub Pages (`gh-pages` branch) on every push or pull request to `master`.
